# Netflix Movies and TV Shows Analysis

## Project Overview

This project performs Exploratory Data Analysis (EDA) on the Netflix Movies and TV Shows dataset to identify trends in content type, release years, countries, ratings, and genres.

## Tools Used

* Python
* Pandas
* Matplotlib
* Seaborn
* Google Colab

## Analysis Performed

* Data Cleaning
* Missing Value Handling
* Duplicate Check
* Exploratory Data Analysis
* Data Visualization
* Insight Generation

## Key Findings

* Movies are more common than TV Shows on Netflix.
* The United States contributes the highest number of titles.
* Netflix content grew rapidly after 2010 and peaked around 2018.
* TV-MA and TV-14 are the most common content ratings.
* International Movies, Dramas, and Comedies are among the most popular genres.

## Files

* `Netflix_EDA_Project.ipynb` – Complete project notebook
* `netflix_titles.csv` – Dataset used for analysis

## Conclusion

The analysis provided valuable insights into Netflix's content distribution, growth trends, audience focus, and genre popularity through data visualization and exploratory analysis.
